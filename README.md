## 整体构架：

主体部分划分为数据曲线、疫情地图两部分

另外还加了时间显示与背景音乐播放器两个配件。


## 运用知识：

用HTML写网页的基础元素

用CSS设置网页的样式和布局，通过外部样式表、内部样式表、内联样式进行不用的样式设计，如分栏、定位、配色等。

用JavaScript实现交互，制作了图片展示与时间显示的部分，并实现了按钮的提交、表格数据显示、、提醒框跳出和地图的切换功能。

用echarts.js实现数据的可视化，通过折线图、地图展示数据。
